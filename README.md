# 💱 Currency Converter

A responsive Currency Converter built using **HTML, CSS, and JavaScript**.  
It fetches real-time exchange rates using the ExchangeRate-API service and performs dynamic currency conversion.

---

## 🚀 Features

- Real-time exchange rates
- Convert between 150+ global currencies
- Dynamic currency selection
- Instant calculation
- Clean UI
- API-based data fetching using fetch()

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- ExchangeRate-API

---
